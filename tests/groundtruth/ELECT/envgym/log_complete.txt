#6 0.794 Packages
#6 0.794 ========
#6 0.794 
#6 0.794 The packages provided here are loosely based on the debian upstream packages with some modifications to make them more usable as non-default pythons and on ubuntu.  As such, the packages follow debian's patterns and often do not include a full python distribution with just `apt install python#.#`.  Here is a list of packages that may be useful along with the default install:
#6 0.794 
#6 0.794 - `python#.#-dev`: includes development headers for building C extensions
#6 0.794 - `python#.#-venv`: provides the standard library `venv` module
#6 0.794 - `python#.#-distutils`: provides the standard library `distutils` module
#6 0.794 - `python#.#-lib2to3`: provides the `2to3-#.#` utility as well as the standard library `lib2to3` module
#6 0.794 - `python#.#-gdbm`: provides the standard library `dbm.gnu` module
#6 0.794 - `python#.#-tk`: provides the standard library `tkinter` module
